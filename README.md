# Форма авторизации с вводом номера телефона и отп кода.

Работа выполнена в рамках тестового задания для прохождения на ШИФТ ЛАБ.

Для запуска проекта в терминале IDE введите :
```code
  npm install
  npm run dev
```


## Использованные технологии
- `axios` 
- `styled-components`
- `vite`
- `react-toastify`
- `react-hook-form`
